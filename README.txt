Como Executar o Programa:
-------------------------
1. Certifique-se de ter imagens de teste na mesma pasta do executável (ou saiba o caminho delas).
2. Execute o arquivo 'main.exe' (presente na pasta principal).
3. Um menu no terminal será exibido com as opções de 1 a 4 correspondentes às questões do trabalho.
4. Digite o número da funcionalidade desejada e siga as instruções na tela (fornecendo o caminho das imagens e parâmetros quando solicitado).

Para rodar a partir do código fonte (opcional):
----------------------------------------------
Requisitos: Python 3.x, OpenCV, NumPy, Matplotlib.
Comando: python main.py

Conteúdo da Entrega:
--------------------
- main.exe (Executável do programa)
- main.py (Código-fonte completo)
- readme.txt (Instruções de uso)
- video_demonstrativo.mp4 (Gravação de tela mostrando o funcionamento)
